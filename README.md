# Coffee-Sales-Dashboard-Analysis
# ☕ Coffee Sales Dashboard Analysis

## 📊 Project Overview

This project presents an **interactive Coffee Sales Dashboard** created using **Microsoft Excel**.
The dashboard helps analyze coffee shop sales performance using different filters such as **Month Name and Day Name**.

It provides insights into:

* Total Quantity Sold
* Total Transactions
* Total Revenue
* Sales by Product
* Sales by Location
* Hourly Sales Trend
* Product Category Performance

The dashboard is **interactive**, meaning users can click on **months or days** to dynamically update the entire dashboard.

The dashboard updates automatically based on the selected month.

### 📆 Day Filter

Users can filter by day:

* Monday
* Tuesday
* Wednesday
* Thursday
* Friday
* Saturday

This allows **day-wise sales analysis**.

# 🚀 How to Use

1. Download the Excel file from this repository.
2. Open **Coffee_Sales_Dashboard.xlsx**
3. Use the **Month slicer** to filter sales by month.
4. Use the **Day slicer** to analyze day-wise sales.
5. All charts will update automatically.
